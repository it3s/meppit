
* Ruby version: 2.0.0

### System dependencies:

* postgresql
* you need to have a execjs runtime (i.e. Nodejs)

### Configuration:

* git clone
* `bundle`
* create and configure your database.yml
* `rake db:create db:migrate`
* `bundle exec rspec` # run tests
* `rails server`  OR  `foreman start` # start server
